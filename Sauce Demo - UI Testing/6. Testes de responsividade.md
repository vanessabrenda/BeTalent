## Critérios analisados
  1. Resposta do sistema a diferentes navegadores
  2. Adaptabilidade a diferentes resoluções de tela
  3. Fluidez ao redimensionar a tela de exibição

### Navegadores verificados
  1. Google Chrome
  2. Microsoft Edge
  3. Mozilla Firefox

### Resoluções verificadas 
 1. Celulares (Mobile): 320px–480px
 2. Tablets: 481px–768px
 3. Laptops: 769px–1024px
 4. Telas grandes: 1024px <
