### Comando para executar o teste de carga com o newman:

    newman run "Restful-Booker - API testing.postman_collection.json" -e "Restful Booker.postman_environment.json" --iteration-count 5         

  Altere o número para escolher a quantidade de carga realizada por vez. O comando acima executará 5 vezes.
    
#### E então, ao pressionar a tecla 'enter' a execução inciará

### Instale o gerador de relatórios para analisar os resultados: 
#### Comando para realizar a instalação do gerador

    npm install -g newman-reporter-html

#### Após executar o comando acima, execute o comando a seguir:
   newman run "Restful-Booker - API testing.postman_collection.json" -e Restful "Booker.postman_environment.json" --iteration-count 5  -r html
